# CRUD_Dapper

CRUD_Dapper é uma aplicação CRUD desenvolvida em ASP.NET Core MVC utilizando Dapper e PostgreSQL.

O projeto permite cadastrar, listar, editar e excluir pessoas através de uma interface web integrada ao banco de dados PostgreSQL.

## Tecnologias Utilizadas

- ASP.NET Core MVC
- Dapper
- PostgreSQL
- Bootstrap
- C#

## Funcionalidades

- Cadastro de pessoas
- Listagem de pessoas
- Atualização de dados
- Exclusão de registros
- Integração com PostgreSQL usando Dapper
